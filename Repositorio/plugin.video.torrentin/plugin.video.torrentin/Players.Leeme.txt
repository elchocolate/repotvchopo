[engine] Motor AceStream - Para reproductor interno (AceStream + Kodi):

Solamente el motor, si se va a instalar tambien el AcePlayer instar primero este, ya
que como el AcePlayer incorpora el motor si se instala primero el AcePlayer luego
no dejará abrir los settings desde este:

version para micros armv7 (la mayoria):
http://dl.acestream.org/products/acestream-engine/android/armv7/latest
version para micros armv7 y x86:
http://dl.acestream.org/products/acestream-engine/android/latest

Xtorrent (Kewlapps)
https://play.google.com/store/apps/details?id=com.gamemalt.streamtorrentvideos
Permite seleccionar directorio para cache y reproductor.

Torrent Player (topl)
https://play.google.com/store/apps/details?id=com.torrent_player
Permite seleccionar directorio para cache pero no reproductor, lleva uno propio compatible con AC3.

Torrent Video Streaming (Itsasoftware)
https://play.google.com/store/apps/details?id=org.itsasoftware.torrent.streaming
Permite seleccionar directorio para cache y reproductor.

Turkey torrent video player (Turkey torrent)
https://play.google.com/store/apps/details?id=rocks.turkeytorrent.player
No permite seleccionar directorio para cache (usa la SD interna) pero permite seleccionar reproductor
y al igual que algunos anteriores recomienda el VLC, mejor usar MXPlayer con codecs AC3

AcePlayer:
Incorpora el motor AceStream, por lo que es RECOMENDABLE instalar primero el
motor solo del primer apartado y despues este, porque si no habra problemas
para abrir los Settings, asi mismo hay que instalar SIEMPRE la misma version del
motor suelto y el AcePlayer:

version para micros armv7 (la mayoria):
http://dl.acestream.org/products/acestream-full/android/armv7/latest
version para micros armv7 y x86:
http://dl.acestream.org/products/acestream-full/android/latest
